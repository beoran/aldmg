ALDMG - ALlegro Direct Mode Gui

Introduction

ALDMG is a direct mode gui library for Allegro 5 written in C, under construction. Direct mode GUIs are much like immediate mode GUI's in API, but they retain more state and defer the drawing to a later stage.
