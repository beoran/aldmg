DMGLFA - a Direct Mode Gui Library For Allegro

Introduction

DMGLFA is a direct mode gui library for Allegro 5 written in C, under construction. Direct mode GUIs are much like immediate mode GUI's in API, but they retain more state and defer the drawing to a later stage.

This project is maintained by beoran
